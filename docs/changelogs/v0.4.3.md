# Changelog v0.4.3

All notable changes to kprun are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.4.3] - 2026-07-11

### Changed

- Deprecated HTTP+SSE transport now emits a warning when used; users are encouraged
  to migrate to the standard Streamable HTTP transport.

### Fixed

- MCP bridge legacy SSE transport fallback narrowed to HTTP 404/405 responses only;
  prevents false-positive fallback when encountering other HTTP error statuses.
- Keyfile unlock error messages clarified for composite keyfile scenarios with
  additional context in error reporting.
