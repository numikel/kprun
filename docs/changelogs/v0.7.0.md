# Changelog v0.7.0

All notable changes to kprun are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.7.0] - 2026-07-21

### Added

- `kprun agents` — install and manage secrets policy blocks for Claude Code agents. Supports auto-detection
  of `AGENTS.md` and `CLAUDE.md` files, with atomic writes and optional global agent visibility.
- `kprun scan` — heuristic committed-secrets checker to detect accidentally committed API keys and
  credentials before they are pushed.

### Changed

- Documentation improvements: clarified project description with problem statement, expanded CLI reference
  with detailed function descriptions, and added comprehensive command documentation.
- Dependency updates: keepass (→0.13.17), clap (→4.6.2), which (→8.0.5), thiserror (→2.0.19),
  keyring (→4.1.5); GitHub Actions taiki-e/install-action (→2.83.4) and softprops/action-gh-release (→3.0.2).
