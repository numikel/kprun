# Changelog v0.6.0

All notable changes to kprun are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.6.0] - 2026-07-16

### Added

- `kprun migrate` — command to migrate vault entries from a legacy KeePass or other password manager
  export format into kprun. Supports structured JSON input with custom field definitions.
- Comprehensive manual testing guide documenting all kprun features, workflows, and edge cases
  for both end users and contributors.
- Windows update troubleshooting section in the README, covering common installation and version
  update scenarios on Windows platforms.

### Changed

- Documentation improvements: clarified master password behavior in Quick start guide, added
  environment variable cleanup examples for testing scenarios, and added migration notes for
  upgrading from v0.5.0.

### Fixed

- `kprun init --quick` now improves credential visibility in the terminal output, making the
  one-time password disclosure clearer and more prominent to the user.
