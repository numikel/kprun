# Changelog v0.6.1

All notable changes to kprun are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [0.6.1] - 2026-07-17

### Changed

- Documentation improvements: documented `cargo install --git` installation method in the README
  quick-start guide for direct installation from the GitHub repository.

### Fixed

- `install.ps1` now works correctly under Windows PowerShell 5.1, resolving execution policy
  and parameter handling issues on older Windows versions.
